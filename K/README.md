# 框架API内容归纳

 ### 目录结构 
 
  > 小程序包含一个描述整体程序的 app 和多个描述各自页面的 page,一个小程序主体部分由三个文件组成，必须放在项目的根目录，如下：
  
  | 文件 | 必填 | 用处 |
  | ------------- |:-------------:| -----:|
  | app.js | yes | 小程序逻辑 |
  | app.json | yes | 小程序公共设置 |
  | app.wxss | no | 小程序公共样式 |

  > 一个小程序页面由四个文件组成，分别是：
  
  | 文件类型 | 必填 | 用处 |
  | js | yes | 页面逻辑 |
  | wxml | yes | 页面结构 |
  | wxss | no | 页面样式表 |
  | json | no | 页面配置 |
  #### 注意：为了方便开发者减少配置项，描述页面的四个文件必须具有相同的路径与文件名。
